## Phase 2 — Repository
Use this repository:

`https://github.com/Unixzap/E1ProjectCalculator`

Upload the CONTENTS of this package to the repository root.

## Phase 3 — Populate repository
On the GitHub repository page:
1. Click **Add file**
2. Click **Upload files**
3. Drag in all files and the `assets` folder
4. Commit directly to the `main` branch

## Phase 4 — Publish with GitHub Pages
1. Open repository **Settings**
2. Select **Pages**
3. Under **Build and deployment**, choose **Deploy from a branch**
4. Select branch `main`
5. Select folder `/ (root)`
6. Click **Save**

Expected website:
`https://unixzap.github.io/E1ProjectCalculator/`

## Application Release
Create a release tagged `v2.3.5` and attach the app ZIP with this exact filename:

`E1-Project-Calculator-Pro.zip`

The website download buttons already use:
`https://github.com/Unixzap/E1ProjectCalculator/releases/latest/download/E1-Project-Calculator-Pro.zip`

## Phase 5 — Custom domain
After buying or choosing the domain:
1. Go to repository **Settings → Pages**
2. Enter the domain under **Custom domain**
3. Click **Save**
4. Add the DNS records GitHub shows at the company where the domain is registered
5. Enable **Enforce HTTPS** when available

Do not create wildcard DNS records.
