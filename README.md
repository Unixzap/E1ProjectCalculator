# E1 Project Calculator Pro Website — Version 3.0

This folder is a complete static website ready for GitHub Pages.

## Upload to GitHub

Replace the current website files in the root of `Unixzap/E1ProjectCalculator` with:

- `index.html`
- `styles.css`
- `script.js`
- the complete `assets` folder

Keep your GitHub Releases and repository settings intact.

## GitHub Pages

In the repository, open **Settings → Pages** and publish from the `main` branch, root folder (`/`).

## Download button

The Download buttons link to:

`https://github.com/Unixzap/E1ProjectCalculator/releases/latest`

When you publish a new GitHub release and mark it as Latest, the website automatically sends users to that release.

## Branding

This website is exclusively for **E1 Project Calculator Pro** and includes no Wet The Face references or assets.
